# 나만의 냉장고 - HTML/CSS/JavaScript 버전

순수 HTML, CSS, JavaScript로 구현된 냉장고 재료 관리 웹 애플리케이션입니다.

## 프로젝트 개요

"나만의 냉장고"는 냉장고 재료를 효율적으로 관리하고, AI 요리 추천을 받으며, 이웃과 식재료 및 배달음식을 공유할 수 있는 커뮤니티 플랫폼입니다.

### 핵심 기능

1. **내 냉장고**: 냉장고 재료 추가 및 유통기한 관리
2. **AI 요리 추천**: 보유한 재료로 만들 수 있는 요리 추천
3. **커뮤니티**: 배달음식 및 식재료 공유/교환
4. **레벨 시스템**: 활동을 통한 경험치 획득 및 레벨업
5. **음식 MBTI**: 나만의 음식 취향 테스트

## 프로젝트 구조

```
html-version/
├── pages/
```

## 실행 방법

### 로컬 서버 사용

ES modules와 로컬 스토리지 접근을 위해 로컬 서버로 실행하는 것을 권장합니다.

**Python 3가 설치되어 있는 경우:**
```bash
cd html-version
python3 -m http.server 8000
```

그 다음 브라우저에서 `http://localhost:8000/pages/index.html` 접속

**Node.js가 설치되어 있는 경우:**
```bash
# http-server 설치 (한 번만)
npm install -g http-server

# 서버 실행
cd html-version
http-server -p 8000
```

그 다음 브라우저에서 `http://localhost:8000/pages/index.html` 접속

**VS Code의 Live Server 확장 사용:**
1. VS Code에서 Live Server 확장 설치
2. `pages/index.html` 파일을 우클릭
3. "Open with Live Server" 선택

## 사용 방법

### 1. 로그인
- `pages/index.html` 페이지에서 시작
- 임의의 이메일과 비밀번호로 로그인 또는 회원가입
- 데이터는 브라우저의 로컬 스토리지에 저장됩니다

### 2. 홈페이지
- 전체 통계 확인 (총 재료, 유통기한 임박, 커뮤니티 게시글)
- 유통기한 캘린더 (현재는 시각적 표시)
- 각 기능으로 빠른 이동

### 3. 내 냉장고
- "재료 추가" 버튼을 클릭하여 새 재료 추가
- 재료명, 카테고리, 수량, 단위, 보관 위치, 유통기한 입력
- 유통기한에 따라 D-Day 배지 표시
- 삭제 버튼으로 재료 제거

### 4. AI 요리 추천
- 사용할 재료 선택 (체크박스)
- "AI 요리 추천받기" 버튼 클릭
- Mock AI가 선택한 재료로 만들 수 있는 레시피 3개 추천
- 각 레시피의 재료, 조리 시간, 난이도, 조리 순서 확인

### 5. 커뮤니티
- "글쓰기" 버튼으로 새 게시글 작성
- 나눔 또는 교환 선택
- 제목, 내용, 거래 지역, 작성자 정보 입력
- 전체/나눔/교환 탭으로 필터링
- 게시글 작성 시 +30 EXP 획득

### 6. 음식 MBTI
- 6개의 질문으로 음식 취향 분석
- 다양한 MBTI 타입 결과 제공
- 주요 특징 및 추천 음식 목록
- 테스트 완료 시 +50 EXP 획득
- 내 정보 페이지에서 결과 확인

### 7. 레벨 시스템
- 활동을 통한 경험치 획득:
  - 재료 추가: +10 EXP
  - AI 요리 추천: +20 EXP
  - 커뮤니티 게시글 작성: +30 EXP
  - 음식 MBTI 테스트 완료: +50 EXP
- 레벨별 필요 경험치: 레벨 × 100
- 실시간 레벨업 알림
- 홈 화면 및 내 정보에서 진행률 확인

### 8. 유통기한 알림
- 헤더의 종 아이콘 클릭
- 3일 이내 유통기한이 임박한 재료 목록 확인
- 유통기한 임박 재료가 있으면 빨간 배지 표시

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - Flexbox 및 Grid 레이아웃
  - 그라데이션 및 색상 활용
  - 반응형 디자인
  - 애니메이션 및 전환 효과
- **JavaScript (ES6+)**:
  - 로컬 스토리지 활용
  - 이벤트 처리
  - DOM 조작
  - 모듈화된 함수 구조

## 데이터 저장

모든 데이터는 브라우저의 로컬 스토리지에 저장됩니다:
- `user`: 사용자 정보
- `ingredients`: 재료 목록
- `communityPosts`: 커뮤니티 게시글
- `userLevel`: 사용자 레벨
- `userExp`: 현재 경험치
- `foodMBTI`: 음식 MBTI 결과

**주의**: 브라우저 캐시를 지우면 데이터가 삭제됩니다.
